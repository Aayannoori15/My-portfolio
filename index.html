Deepfake Detector (Real vs Fake Images)

PyTorch + Django

A web app that detects real vs fake images using a custom CNN served via Django. Users upload an image, get a live preview, and receive a verdict plus P(fake).

  • In-memory inference only (no file or database storage).
  • Preprocesses to 32x32 RGB and scales to [0,1].
  • CNN outputs a sigmoid probability P(fake) for the final verdict.
